# Calculadora em C

Programa simples em C que realiza operações matemáticas básicas diretamente no terminal.

## Funcionalidades
- Adição  
- Subtração  
- Multiplicação  
- Divisão (com verificação de divisão por zero)  
- Opção para sair do programa

## Como compilar e executar

## Usando GCC:
```bash
gcc main.c -o calculadora
./calculadora

Entrada e saida de dados
=== Calculadora ===
1 - Adição
2 - Subtração
3 - Divisão
4 - Multiplicação
0 - Sair
Escolha uma opção: 1
Digite o primeiro número: 5
Digite o segundo número: 3
Resultado: 8.00
